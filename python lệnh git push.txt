xem danh sách repo đang dùng để push

1️⃣ Cấu hình cho toàn bộ máy tính (global) — dùng 1 lần là xong

Gõ vào PowerShell hoặc CMD:

git config --global user.name "Tên của bạn"
git config --global user.email "email_github_của_bạn@example.com"
